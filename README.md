# FORMLÆRE, program for norsk industridesign

> Å gjere for norsk industridesign det Snøhetta gjorde for norsk arkitektur:
> gje faget ein grammatikk, ei estetisk line og ei røyst, frå det djupaste
> idégrunnlaget til objektet du kan printe i dag.

Dette repoet er **maskinlesbar spegel og loop-motor** for prosjektet.
Sanninga for menneske bur i Notion; sanninga for agentar bur her.

## Struktur

| Fil/mappe | Rolle |
|---|---|
| `CLAUDE.md` | Harness-instruks: kva ein agent-sesjon treng å vite |
| `LOOP.md` | Loop-protokollen: den autonome, uendelege arbeidssyklusen |
| `STATE.md` | Loop-state: iterasjonsteljar, sist gjort, neste handling |
| `BACKLOG.md` | Prioritert arbeidskø som loopen konsumerer og fyller på |
| `LEARNINGS.md` | Ratchet: kvar feil vert til ein regel, éin gong |
| `notion/manifest.json` | Alle Notion-ID-ar (sider, databasar, datakjelder, views) |
| `strategy/` | Markdown-speglar av strategisidene i Notion |
| `research/` | Daterte research-notat som loopen produserer |

## Tesen i éi setning

**Form follows fitness.** Industridesign er den ærlege signaturen av
constraint: objekt som ber sine tusen avgjerder leseleg, optimerte over
mange aksar i staden for éin, laga for å vare og bli slitne vakkert,
forma på vegner av dei som skal leve med forma.

## Kjelder

Fundamentet er traktaten *FORMLÆRE, Ei traktatform om form for design og
estetiske fag* (Finne, AHO 2026), Designopprøret-manifestet og diktsyklusen
*Kva er design?*. Notion-prosjektet: sjå `notion/manifest.json`.
